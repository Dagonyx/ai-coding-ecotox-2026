### In course

1. Make sure all students have R and Positron or R Studio installed
2. Make sure all students have a github account and access to github
3. Check in - who has coded before? Who has no coding experience?
(Team up people (groups of 2 - 3) according to their skills)
4. Set github configs with students via:
    ```bash
    git config --global user.name GITHUB_NAME
    git config --global user.email GITHUB_EMAIL
    ```
    In case of proxy settings use:
    ```bash
    git config --global http.proxy PROXY_LINK
    git config --global https.proxy PROXY_LINK
    ```

5. Clone the coding sandbox repo: https://github.com/hreinwald/ai-coding-ecotox-2026

    Make sure you can access the link above. If you already have a dedicated github repository on your system feel free to use that and clone the repo into it. If not create a `github` folder somewhere central on your device. You can do this simply via: 
    ```bash
    mkdir ~/github
    cd ~/github
    ```

    You can clone the repository simply through the Positron IDE or by executing the following lines in your git console: 
    ```bash
    git clone https://github.com/hreinwald/ai-coding-ecotox-2026
    cd ai-coding-ecotox-2026
    ```
