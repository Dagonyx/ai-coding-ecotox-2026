## Translate .Rmd to .qmd with AI

To get you warmed up with agentic coding do a simple task using your IDE's KI-tool. 

### Tasks

1. Tell the AI to translate the `dose-response-workflow.Rmd` file for you into a Quarto Markdown file. Make sure to tell the AI where to find the file and where to store it (ideally in a new folder called `qmd`). Before you get your agent to work use the *PLANING* mode first to get your AI assistant warmed up for the task. 

2. Once the Quarto Markdown is created render both documents (the `.Rmd` and the `.qmd`) to see if any errors occure. If you don't know how to do that - ask your AI assistant.

3. Inspect and compare both html reports. Did the KI introduce any mistakes or errors? If so - where do they might come from? 

### INFO

For more insights on [R Markdown](https://rmarkdown.rstudio.com/) and [Quarto](https://quarto.org/), refer to their official documentation.
