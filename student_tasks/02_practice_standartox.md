## Get to know standartox

### Task

This is a simple code along task. Open the `standartox-workflow-example.Rmd` file to get started.

You can render the file to an html if you wish to have a nicer overview. If not feel free to dive directly into the code and work your way through the script **step-by-step**. 

To execute the code chunks simply click on `Run Cell`. If you wish to execute specific lines mark them or click on that line then hit `Ctrl + Enter` to run them in your console.

If you run into problems try to solve it using your AI assistant by clicking on `Fix` or `Explain` (in case of errors).

### Bonus

* Have a look at the files in the `llm_context` directory and its files. What do you think might be the reason for their existence? 

* Have a chat with your AI assistant (planning mode) on what might be in the standartox data base for you to explore. You can help your AI by telling it to first look at the `llm_context/standartox_data_dictionary.json` file. Any research question you might come up with? (e.g. How well do the endpoints between *Ceriodaphnia* and *Daphnia* match?)

* Design a prompt for your AI agent that would build such a `standartox_data_dictionary.json`. Name yours differently of course. ;) Run it and see if both files describe the same context.
